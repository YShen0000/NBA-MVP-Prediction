Codebook:- `Rank`: MVP Rank- `Player`: Player's Name- `Age`: Player's aGE- `Tm`: Team- `First`: First Place Votes- `Pts.Won`: Points won- `Pts.Max`: Maximum Points- `Share`: Points Share- `G`: Games Played- `MP`: Minutes Played per Game- `PTS`: Points per Game- `TRB`: Total Rebounds per Game- `AST`: Assist per Game- `STL`: Steals per Game- `BLC`: Blocks per Game- `FG.`: Field Goal Percentage- `X3P.`: 3-point Field Goal Percentage- `FT.`: Free Throw Percentage- `WS`: Win Shares- `WS.48`: Win Shares per 48 Minutes- `Year`: Year of MVP earned
